Barcode server

Parameters

value=value for the code to be generated
scale=scale for the image

Examples

EAN13

/ean13?value=7840058004215
/ean13?value=7840058004215&scale=2

CODE39

/code39?value=12345678
/code39?value=12345678&scale=2

CODE128

/code128?value=98380d7d-85df-456f-901d-3b325ee86e07
/code128?value=98380d7d-85df-456f-901d-3b325ee86e07&scale=2

PDF417

/pdf417?value=patricio+antonio+diaz+gimenez
/pdf417?value=patricio+antonio+diaz+gimenez&scale=2

QR

/qr?value=98380d7d-85df-456f-901d-3b325ee86e07
/qr?value=98380d7d-85df-456f-901d-3b325ee86e07&scale=2
